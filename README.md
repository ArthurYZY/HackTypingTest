# HackTypingTest

一个小玩具，用于自动在https://barneyzhao.gitee.io/typing-cn/#/monkey进行打字测试。

想看看高WPM的朋友们打字是什么样的。。。。

### 依赖

* python >= 3.6
* selenium
* Chrome浏览器

### 使用方法

下载相应的chrome驱动到项目文件夹下

在命令行中执行

```python
python3 typing.py
```



### PS

1. 代码中的WPM可以任意修改，由于延迟等问题，可能会和网站上统计的略有误差。
2. （这个网站的WPM可以是infinity的）
